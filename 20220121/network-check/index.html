# 主体思路

  • 检查物理链路

  • 查看本机 IP 地址、路由、DNS 的设置；

  • 测试网关或路由器的通畅情况。先测网关然后再测路由器，一级一级地测试；

  • 测试 ping 公网 IP 的通畅情况

  • 测试 DNS 的通畅情况
