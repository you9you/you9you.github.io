# 解决方案

# 清除缓存

根据日志的标识查找文件"b3542d63898d9493a047ab9ab041d76b"

/jellyfin/config/metadata/library/b3/b3542d63898d9493a047ab9ab041d76b

将ass字幕文件删除

# 调整字幕

使用Aegisub格式化(另存文件即可)

# 刷新

刷新元数据即可

# 其他问题
